# Artikel Bela Negara

Repository ini berisi artikel mahasiswa dengan judul **"Peran Bela Negara dalam Menjaga Keutuhan dan Kedaulatan Indonesia"**.

Artikel ini disusun untuk memenuhi tugas mata kuliah dan membahas pengertian, tujuan, serta penerapan bela negara dalam kehidupan sehari-hari, termasuk tantangan bela negara di era digital.

## Isi Repository
- Artikel dalam format dokumen (PDF / DOCX)
- File pendukung jika diperlukan

## Tujuan
Memberikan gambaran mengenai pentingnya peran warga negara, khususnya generasi muda, dalam menjaga persatuan dan ketahanan nasional.
